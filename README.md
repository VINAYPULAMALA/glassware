# Glassware Catalog

A beautiful, responsive product catalog for glassware with 194 products across 9 categories.

## Features

- **Complete Product Catalog**: All 194 products from the ODT/PDF document
- **Dropdown Category Filter**: Select specific categories from a dropdown menu
- **Real-time Search**: Search products by name instantly
- **Live Statistics**: Shows current number of visible products and categories
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Smooth Animations**: Hover effects and smooth scrolling
- **Back to Top Button**: Quick navigation back to the top

## Files

- `index.html` - Main HTML file with all products
- `styles.css` - Stylesheet with modern, responsive design
- `script.js` - JavaScript for filtering and interactivity
- `images/` - Folder containing all product images
- `380ml-tikishots-oval-block-beer-mug.odt` - Original product data (ODT format)
- `380ml-tikishots-oval-block-beer-mug.pdf` - Original product data (PDF format)

## How to Use

1. **Open the Catalog**: Simply open `index.html` in your web browser
2. **Browse Products**: Scroll through all categories
3. **Filter by Category**: Use the dropdown menu to show only specific categories
4. **Search Products**: Type in the search box to find specific products
5. **View Details**: Each product card shows:
   - Product image
   - Product name
   - Pack size (pieces)
   - Price in INR

## Technical Details

- Pure HTML, CSS, and JavaScript (no frameworks required)
- Grid layout for responsive product display
- Fallback images for missing product photos
- Modern gradient design with smooth transitions
- Cross-browser compatible

## Pricing Note

All prices are displayed **per piece** in INR (Indian Rupees). Prices shown are 3x the original wholesale prices.
